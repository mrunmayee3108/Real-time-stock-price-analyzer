# Real-time Stock Price Analyzer

A simple Python project that fetches real-time stock data, visualizes recent price trends, and provides quick analytical insights such as average, highest, and lowest prices.


## Features
- Fetches live stock data using Yahoo Finance API (`yfinance`)
- Plots closing prices of the past month
- Displays key statistics:
  - Average price
  - Highest price
  - Lowest price


## Tech Stack
- Python 3+
- yfinance – for stock data
- matplotlib – for visualization
- pandas – for data handling
